# 마잼네컷

🔗 URL : https://4cut-photo.netlify.app/

## 01. 프로젝트 소개

마천동 성당에서 진행한 어린이날 행사 사진을 인생네컷으로 만들어주는 '마잼네컷' 프로젝트입니다.

### 실행

```
$ git clone https://github.com/jisooround/4cut_for_majam.git
```

```
$ npm install
```

```
$ npm run dev
```
